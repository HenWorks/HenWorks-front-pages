Opclaw
OpenClaw on Android

AI agent terminal for your Android device — phone, tablet, or TV. Run AI agents locally, no cloud dependency.

Google Play Source Code

IMPORTANT: Opclaw is an independent, unofficial client. It is NOT developed, endorsed, or affiliated with the official OpenClaw project.

Key Features
🤖

AI Agent Terminal

Run AI agents powered by your preferred LLM provider directly from your device. Support for Anthropic, OpenAI, and custom API endpoints including local models.

🌐

Local Gateway

Built-in gateway server lets you interact with AI agents through a modern web interface. Configure once, access anytime via your browser.

🛠️

Developer Tools

Install powerful command-line tools: jq (JSON processor), ripgrep (ultra-fast search), gh (GitHub CLI), tmux (terminal multiplexer), and more.

📱

Multi-Device Support

Optimized for phones, tablets, and Android TV. The same AI terminal experience across all your Android devices.

⚡

Easy Setup

One-tap setup with guided onboarding. Choose your AI provider, configure your API key, and start working in minutes.

🔄

In-App Updates

Check for and apply OpenClaw platform updates directly from Settings — no manual terminal commands needed.

How It Works

Install

Install Opclaw and complete the quick setup wizard

Connect

Connect your preferred AI provider (API key required)

Launch

Start the gateway and open the web interface

Work

Run AI agents, manage conversations, and use developer tools

Requirements
  • Internet connection for AI API calls
  • API key from your AI provider (Anthropic, OpenAI, or compatible)
  • Android 10 or higher

主要功能
🤖

AI 代理終端

直接在裝置上運行由你偏好的 LLM 提供者驅動的 AI 代理。支援 Anthropic、OpenAI 及自訂 API 端點，包括本地模型。

🌐

本地網關

內建網關伺服器，讓你透過現代化的網頁介面與 AI 代理互動。一次設定，隨時透過瀏覽器存取。

🛠️

開發者工具

在裝置上直接安裝強大的命令列工具：jq（JSON 處理器）、ripgrep（超快速搜尋）、gh（GitHub CLI）、tmux（終端多工器）等。

📱

多裝置支援

針對手機、平板和 Android TV 最佳化。在所有 Android 裝置上享有一致的 AI 終端體驗。

⚡

簡易設定

一鍵設定搭配引導式流程。選擇 AI 提供者、設定 API 金鑰，幾分鐘內即可開始使用。

🔄

應用內更新

直接從設定中檢查並套用 OpenClaw 平台更新，無需手動輸入終端指令。
